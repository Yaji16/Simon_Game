Welcome to the Simon Challenge game!

This game tests your memory skills! Are you up for it?

Instructions:

  • Press any key to start the game.
  • Observe the button that has been highlighted.
  • Click on it to get to the next level.
  • Another button gets highlighted.
  • Click on the button you clicked on the starting level and then the button highlighted recently.
  • If the sequence is properly followed you keep progressing to the next level!

Good Luck! Go ahead and play now!
